## 오늘 작업 내용

### 1. Google Sheets API 연동 문제 해결
- 오류: `Google Sheets API has not been used in project 392521991205`
- 원인: provider_token이 서버 세션에서 누락되는 Supabase SSR 한계
- 해결: 클라이언트에서 provider_token 직접 획득 후 API route로 전달
- 추가: GOOGLE_API_KEY 환경변수 방식 도입 (공개 시트 대상)
- Vercel 환경변수 추가: `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, `GOOGLE_API_KEY`

### 2. Groq 토큰 한도 초과 문제 해결
- 오류: 413 tokens per minute limit exceeded (요청 24,333 / 한도 12,000)
- 해결: 최근 10행으로 제한 + 셀 100자 초과 시 트런케이트

### 3. 인사이트 품질 개선
