## 수정 2 — 수습/인턴 평가 차단 로직 개선

### 증상
수습 종료가 26년 6월인 경우, H2(하반기) 평가가 막히는 현상.

### 원인
`getEvalBlockedInfo()`가 분기 첫 달 기준으로 차단 여부를 판단할 때 `currentHalfIdx`(평가 페이지 변수)를 하드코딩해 사용.
대시보드에서 다른 반기(`currentDashboardHalf`)로 조회 시 엉뚱한 반기 기준으로 판단.

### 수정
`getEvalBlockedInfo(memberId, halfIdxOverride)` — 옵셔널 `halfIdxOverride` 파라미터 추가.
- 대시보드 → `currentDashboardHalf` 전달
- 팀리더 평가 페이지 → `currentHalfIdx` 전달
- 수습 종료 월이 H2 첫 분기 시작 이전이면 차단 해제

커밋: `a99d8c5`, `ce2d7d3`

## 기능 1 — 팀 취급고 현황 탭 구조 개편 + 수수료 수익 탭 추가

### 탭 구성 (순서)
1. **전체보기** (신규, 기본 탭) — 월별 취급고 + 수수료 수익 + 재직자 한 화면
2. **월별 취급고** (기존)
3. **수수료 수익** (신규) — 직접 수기 입력
4. **재직자** (기존)

### 수수료 수익 탭 기능
- 1팀 / 2팀 월별 수수료 수익 직접 입력 (만원 단위)
- 행 합계 / 열 합계 / 연간 합계 실시간 자동 계산
- **1팀원(수) / 2팀원(수)** 월별 팀원 수 입력 (입사·퇴사 반영)
- **평균 인당 수익** 자동 계산 (1팀원+2팀원 합산 기준)
- 🔍 메모 버튼 — 셀별 팝업 메모 저장
- **증빙 버튼** — 1팀 / 2팀 Google Sheets 링크 연결 (섹션 타이틀 우측)
- `feeRevenue[fyId][team][month]` / `headcount1/2[month]` / `headcountMemo1/2[month]` 구조로 저장
- `_buildStateObj` / `_restoreStateObj` 에 `feeRevenue`, `profileDisplayOrder` 반영

커밋: `69c21c3`, `7b826f4`, `b90a0b6`, `12d0f6c`, `cb5593a`, `b23d026`, `ef23051`, `24bb3bf`

## 기능 3 — 팀리더 평가 페이지 분리

### 개요
직책이 `팀리더`인 팀원은 일반 매니저 평가 레이아웃 대신 별도 페이지 표시.

### 구현
- `showEvalForMember()` — 팀리더 분기: `eval-layout` 숨김 + `leader-eval-container` 표시
- `renderLeaderEvalPage(m)` 신규 함수:
  - 수수료 수익 테이블 (읽기 전용, `forceReadonly=true`)
  - 해당 팀 소속 매니저 평가 현황 테이블 (현재 반기 기준)
  - 팀 요약 카드 (전체 팀원 수 / 평가 완료 / 임시저장 / 팀 평균)
  - 평가 대상 아님 인원 붉은색(`⛔ 평가 대상 아님`) 표시

- 팀리더는 사이드바 서브메뉴에는 정상 표시 (기존 제외 조건 원복)

커밋: `7294c17`, `795e42f`, `3905270`, `fcfd52a`, `5da8d68`, `c52dc10`, `636e312`

## 기능 4 — 대시보드 H1/H2 반기 탭

### 개요
대시보드에 H1 상반기 / H2 하반기 탭 추가. 각 반기별 제출·임시저장 데이터만 표시.

### 구현
- `currentDashboardHalf` 변수 (0=H1, 1=H2) 추가
- `setDashboardHalf(h)` 함수로 탭 전환 + 재렌더
- `submittedReports[memberId].halfIdx` / `draft.halfIdx` — 저장 시 반기 태그 자동 부여
- 기존 halfIdx 없는 데이터는 `?? 0`으로 H1 귀속 처리

커밋: `c52dc10`, `593ff1e`
