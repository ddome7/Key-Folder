# 드래그앤드롭 순서 변경 기능 개발내역

> 작성일: 2026-04-06
> 커밋: `f895b18`

---

## 작업 배경

팀원 목록과 설정 가입인원 목록의 순서를 관리자가 자유롭게 변경할 수 있도록 요청.
초기 구현은 ▲▼ 버튼 방식이었으나, 마우스 드래그 방식으로 전환.

---

## 구현 내역

### 1. 공통 드래그앤드롭 유틸 (`_dndSetup`)

HTML5 Drag and Drop API 기반 공통 함수 추가.

- `dragstart` → 원본 행 반투명(opacity 0.4)
- `dragover` → 드롭 대상 행 파란 하이라이트 (`.dnd-over`)
- `drop` → 콜백(`onDrop`) 호출
- `dragend` → 스타일 초기화

### 2. 팀원 목록 드래그앤드롭 (`initMemberDragDrop`)

- `memberRowHtml()` 수정: `<tr draggable="true" data-member-id data-group>` 추가
- 각 행 첫 번째 `<td>`에 드래그 핸들 아이콘 `⠿` 추가
- **같은 그룹(팀+상태)** 내에서만 이동 허용, 다른 그룹 간 드롭 무시
- drop 후 DOM 순서 → `TEAM_MEMBERS` 배열 동기화 → `saveAppState()` 저장
- `renderMemberList()` 마지막에 `initMemberDragDrop()` 자동 호출

### 3. 설정 가입인원 드래그앤드롭 (`initProfileDragDrop`)

- `renderSettingsPage()` 내 approved row에 `draggable="true" data-profile-id` 추가
- drop 후 새 DOM 순서를 `profileDisplayOrder` 배열(UUID 목록)에 저장
- `saveAppState()` → Supabase + localStorage에 영구 저장
- 다음 `renderSettingsPage()` 호출 시 `profileDisplayOrder` 기준으로 목록 정렬

### 4. 상태 저장 확장

`_buildStateObj` / `_restoreStateObj`에 `profileDisplayOrder` 필드 추가.

### 5. CSS

```css
.dnd-handle { color: var(--gray-300); cursor: grab; ... }
tr[draggable="true"] { cursor: default; }
tr.dnd-dragging { opacity: 0.4; }
tr.dnd-over td { background: #dbeafe !important; }
```

- `body.viewer-mode .dnd-handle { display: none }` — 뷰어는 순서 변경 불가
