## 기능 4 — 팀장 평가 페이지 실시간 등급 계산 카드

### 구성
| 항목 | 가중치 | 기준 |
|------|--------|------|
| ① 매출 강화 | 70% | 평균 인당 수익 기준 A/B/C |
| ② 팀 성장 | 30% | 팀원 평균 등급 기준 A/B/C |
| 최종 추정 등급 | — | 가중 합산 결과 |

### 매출 강화 등급 기준
- A: 평균 인당 수익 > 13,000,000원 (1,300만원)
- B: 9,750,000원 ≤ 평균 인당 수익 ≤ 13,000,000원 (975만원~1,300만원)
- C: 평균 인당 수익 < 9,750,000원 (975만원 미만)

### 최종 등급 계산
- A=3, B=2, C=1 포인트 환산
- 가중점수 = 매출강화점수×0.7 + 팀성장점수×0.3
- ≥2.5 → A / ≥1.5 → B / <1.5 → C

### 수정 이력
- **초기**: 월평균 총수익으로 비교 → 잘못된 등급
- **수정**: 평균 인당 수익(연간합계) 기준으로 변경
  - 공식: 반기 총수익 ÷ 반기 전체 월별 인원수 합산

커밋: `a10a13e`, `088bed5`
